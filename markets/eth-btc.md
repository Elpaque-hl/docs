# ETH/BTC

Tracks the price ratio of ETH to BTC.

**Current ratio:** ~0.028

## Specifications

| Parameter | Value |
|-----------|-------|
| Underlying | `price(ETH) / price(BTC)` |
| Max leverage | 10x |
| Initial margin | 10% |
| Funding | Hourly |
| Settlement | USDC |

## Use cases

* **Macro rotation** — The most watched ratio in crypto. Long if ETH outperforms, short if BTC dominance rises.
* **Cross-asset hedging** — Isolate ETH alpha from BTC beta on an ETH long
* **Single-position pair trade** — One margin, one liquidation based on the ratio
