# HYPE/SOL

Tracks the price ratio of HYPE to SOL.

**Current ratio:** ~0.12

## Specifications

| Parameter | Value |
|-----------|-------|
| Underlying | `price(HYPE) / price(SOL)` |
| Max leverage | 10x |
| Initial margin | 10% |
| Funding | Hourly |
| Settlement | USDC |

## Use cases

* **Hyperliquid vs Solana** — Trade HYPE outperformance against the leading alt-L1
* **L1 rotation** — Capture capital flows between Hyperliquid and Solana ecosystems
* **Single-position pair trade** — No need to short SOL separately and risk getting liquidated on it
