# HYPE/BNB

Tracks the price ratio of HYPE to BNB.

**Current ratio:** ~0.028

## Specifications

| Parameter | Value |
|-----------|-------|
| Underlying | `price(HYPE) / price(BNB)` |
| Max leverage | 10x |
| Initial margin | 10% |
| Funding | Hourly |
| Settlement | USDC |

## Use cases

* **Hyperliquid vs BNB Chain** — Trade HYPE outperformance against the largest CEX ecosystem token
* **DeFi vs CeFi** — Express a view on decentralized infrastructure vs centralized exchange dominance
* **Single-position pair trade** — No need to short BNB separately and risk getting liquidated on it
